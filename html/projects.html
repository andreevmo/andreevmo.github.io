Проекты:

  • "Игры разума”

    Описание:
    Консольное приложение состоящее из пяти игр-викторин, где пользователю нужно для успешного завершения ответить верно на 3 вопроса.
    Игры: проверка на четность, является ли число простым, решить арифметическое выражение, заполнить недостающий элемент прогрессии, найти наибольший общий делитель.
    Полученные навыки:
    Работая над проектом закрепил навыки работы с Java core, Git (Github), Gradle, научился настраивать автоматическую проверку сервисом codeclimate на качество кода.

  • "Вычислитель отличий"

    Описание:
    Консольное приложение умеющее сравнивать два файла с данными в формате yaml или json.
    Полученные навыки:
    Работая над проектом закрепил навыки работы с JUnit 5, научился работать с файлами, структурами данных, Jakson, Github Actions, настроил автоматическую проверку сервисом codeclimate на покрытие тестами.

  • "Валидатор данных”

    Описание:
    Библиотека для осуществления проверки объектов на соответствие заданным требованиям. В текущей реализации может проверять целые числа, строки и map’ы.
    Полученные навыки:
    Работая над проектом закрепил навыки работы с ООП.

  • "Анализатор страниц"

    Описание:
    Приложение позволяющее проверить сайт на СЕО-пригодность.
    Полученные навыки:
    Проект является веб-сайтом построенным на фреймворке Javalin, ORM Ebean, База PostgreSQL для продакшена + для разработки H2. Для рендеринга использовал Thymeleaf, для фронтенда использовал Bootstrap. Автотесты на Junit 5, Unirest-Java, Mockito.
